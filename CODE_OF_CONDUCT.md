# 行为准则

## 我们的承诺

我们致力于为每一位参与者营造无骚扰的协作体验，无论其性别、年龄、种族、宗教、能力或技术背景如何。我们承诺让社区成员在贡献、提问与讨论时感到安全、受到尊重并被倾听。

## 我们的标准

有助于营造积极环境的行为示例包括：

* 展现友善与同理心。
* 尊重不同观点与经历。
* 建设性地接受反馈，并聚焦于项目目标。
* 关注社区整体福祉。

不可接受的行为示例包括：

* 使用带有歧视、骚扰或暴力倾向的言语或图像。
* 人身攻击或侮辱性评论。
* 公共或私下的骚扰。
* 未经明确许可的敏感信息披露。
* 其他被认定为不专业或不恰当的行为。

## 执行责任

项目维护者负责解释与执行行为准则，并采取适当行动回应任何不可接受的行为。维护者有权利与义务删除、编辑或拒绝违背本准则的评论、提交、代码、wiki 编辑、issue 或其他贡献。

## 举报

若你遭遇或目击不当行为，请通过以下方式联系维护团队：

* 邮件：`maintainers@example.com`

维护者将审查每一份举报，并在合理时间内给出回应。我们承诺尊重举报人的隐私与安全。

## 适用范围

本准则适用于所有项目空间（包括 issue、讨论区、代码仓库、线下活动等）以及当个人代表项目或社区与公众交流时。

## 执行指南

1. **纠正**：若行为被认为不恰当但影响较轻，维护者将发布非公开或公开的警告，并澄清期待的行为标准。
2. **警告**：若行为持续或严重，维护者可能发布正式警告，并要求停止违规行为。
3. **临时或永久禁令**：对于重复或严重违规者，维护者可依据情况限制其参与项目的权限。
