# 项目目录索引

本索引帮助新成员快速了解 AOS（Agent Operating System）仓库的规划结构与各子目录职责。实际代码仍在搭建过程中，如目录尚未出现，请参考“状态”列判断其落地情况。

## 顶层目录速览

| 目录        | 说明                                                               | 状态      |
| ----------- | ------------------------------------------------------------------ | --------- |
| `AGENTS.md` | 面向 AI/人类协作者的操作手册，定义默认工作流、质量门禁与守护策略。 | ✅ 已提供 |
| `README.md` | 仓库概览，将指向文档索引与快速导航入口。                           | ✅ 已提供 |
| `docs/`     | 设计、操作、流程等文档合集（见下文子目录）。                       | ✅ 已提供 |
| `kernel/`   | 事件总线、内存、守护与度量的微内核实现。                           | ⏳ 规划中 |
| `plugins/`  | 感知器（perceiver）、规划器、执行器、评审器、修补器等插件化组件。  | ⏳ 规划中 |
| `packages/` | 公共包：provider/router、agent registry、prompts、schema 定义。    | ⏳ 规划中 |
| `apps/`     | 前端/服务端应用（Next.js 控制台、API 服务等）。                    | ⏳ 规划中 |
| `skills/`   | 轻量本地技能（如 `csv.clean`、`stats.aggregate`、`md.render`）。   | ⏳ 规划中 |
| `tests/`    | 单测、集成测试与回放用例集合。                                     | ⏳ 规划中 |
| `episodes/` | 运行轨迹（append-only JSON），用于回溯与评分。                     | ⏳ 规划中 |
| `reports/`  | 每次运行生成的交付报告（Markdown/HTML）。                          | ⏳ 规划中 |

> ✅ 表示目录已存在；⏳ 表示规划中，可在后续迭代中创建；🛠️ 表示建设中。

## `docs/` 子目录

| 子路径                              | 说明                                              |
| ----------------------------------- | ------------------------------------------------- |
| `docs/README.md`                    | 文档首页，列出主要知识入口。                      |
| `docs/目录索引.md`                  | 本文件，解释仓库目录与职责。                      |
| `docs/详细设计与API契约.md`         | 核心架构、模块职责、API 契约与运行流程说明。      |

## 新成员入门建议

1. **阅读 `AGENTS.md`**：了解团队流程、工具约束与交付标准。
2. **浏览 `docs/目录索引.md`（即本文）**：建立对仓库骨架的整体认知。
3. **研读 `docs/详细设计与API契约.md`**：掌握架构、模块职责与端到端流程。
4. **按需参照 `docs/操作指南_Episodes与Replay.md`**：亲手跑通最小闭环，熟悉事件日志体系。
5. **回顾 `docs/需求草稿/`**：补齐背景故事、目标用户与演进路线。

如需进一步信息，可在对应目录下的 `AGENTS.md` 或 `README` 中查阅专属约束与最佳实践（创建目录时建议同步补充）。
