# Episodes 页面手动验收记录

- 骨架屏：首次进入 `/episodes`，在网络节流为 Slow 3G（Chrome DevTools）下可见列表与详情骨架，占位时间约 650ms。
- 错误 Toast：断开 `pnpm dev` 后刷新页面触发加载失败出现底部 Toast，点击“重试”后在服务恢复时成功拉起列表。
- 回放按钮：选择最新 Episode，点击“触发回放”后约 120ms 内展示回放结果条，显示原始/回放分数与差值 0。
- 可达性：列表单元一键选中即可展示详情，回放只需额外一次点击，关键流程 ≤ 3 次点击。

> 录屏与 Web-Vitals 采集请使用浏览器性能面板，操作脚本参考 `docs/操作指南_Episodes与Replay.md`。
